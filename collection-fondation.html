Le premier parcours

Tout grand progrès commence par une fondation solide.

Cette première collection développe progressivement les qualités physiques essentielles. Chaque entraînement possède son propre objectif, mais les mouvements détaillés restent dans les vidéos afin de préserver l’expérience guidée ZifaFit.

Gratuit

Les vidéos

Découvre les séances, les démonstrations et les adaptations directement sur YouTube.

Optionnel

Les cartes officielles

Un format clair et imprimable sera offert pour les personnes qui veulent s’entraîner avec les cartes.
