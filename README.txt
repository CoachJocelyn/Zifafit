ZIFAGONG — BANNIÈRE AVEC MENU

Fichiers inclus :
- zifagong.html
- zifa-gong-banner.css
- zifa-gong-banner.png

Installation :
1. Mets les trois fichiers dans le même dossier que ta page.
2. Ouvre zifagong.html pour tester.
3. Adapte seulement les liens du menu si tes noms de pages sont différents.
4. La page Zifa Gong est déjà sélectionnée en vert.
5. Il n’y a aucun bouton ajouté sur l’image et aucun espace blanc entre le menu et la bannière.

Pour intégrer dans une page existante :
- copie la section <section class="zifa-hero">...</section>
- copie le contenu du fichier CSS
- garde le petit script du menu mobile
